# Davin Zaki Pradipa - Portfolio CV

Proyek ini adalah Curriculum Vitae (CV) interaktif berbentuk _landing page_ yang dibangun menggunakan teknologi web modern.

## 🚀 Teknologi yang Digunakan

- **Vite** - Build tool.
- **Three.js** - Menampilkan _background_ partikel 3D yang interaktif terhadap pergerakan _mouse_.
- **GSAP** - _GreenSock Animation Platform_ untuk efek transisi dan animasi tulisan/konten yang smooth.
- **Vanilla JavaScript & CSS** - Styling.
- **Catppuccin Mocha** - Sebagai referensi _color palette_.

## 📋 Persyaratan (Prerequisites)

Pastikan Anda sudah menginstal **Node.js** di komputer Anda. Anda bisa mengunduhnya dari [nodejs.org](https://nodejs.org/).

## 🛠️ Cara Install & Menjalankan (Local Development)

1. Buka terminal dan pastikan Anda berada di direktori proyek ini.
2. Instal semua dependensi yang dibutuhkan dengan menjalankan:
   ```bash
   npm install
   ```
3. Setelah instalasi selesai, jalankan server _development_ (Vite):
   ```bash
   npm run dev
   ```
4. Buka _browser_ Anda dan akses URL yang tertera di terminal (biasanya `http://localhost:5173/`).
